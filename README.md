# README

# 1- Instalar las dependencias de Laravel y php
```
composer install
```

# 2- Instalar las dependencias de node.js con npm
```
npm i
```
# 3- Correr las migraciones
```
php artisan migrate
```
# 4- Levantar proyecto en forma local
```
php artisan serve
```

# 5- Compilar los componentes 
```
npm run dev
```
